# 📋ToDoList

날짜를 선택해 원하는 날짜에 할일 목록을 추가 및 삭제할 수 있는 어플리케이션

- 개발 환경 : Android Studio

# ⚙️ 기능

---

- 사용자가 할일을 입력하고자 하는 날짜를 선택할 수 있습니다.
- 해당 날짜에 할일 목록을 추가하거나 수정할 수 있습니다.
- 할일 목록은 스크롤뷰로 관리됩니다.
- 수정 단계에서는 삭제 기능을 사용할 수 있습니다.
- 할일을 마친 목록에 대해서는 체크 박스를 통해 표시할 수 있습니다.
